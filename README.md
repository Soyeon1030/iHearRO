# margin & padding 

m(숫자) - margin 

mt - margin top

mb - margin bottom

ml - margin left

mr - margin right

mx - margin left & margin right

my - margin top & margin bottom

---

p(숫자) - padding 

pt - padding top

pb - padding bottom

pl - padding left

pr - padding right

px - padding left & padding right

py - padding top & padding bottom
